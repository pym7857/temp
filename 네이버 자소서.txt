[웹 프로그래밍 분야의 다양한 프로젝트 경험]
대학교 2학년 겨울방학때, 처음으로 웹 프로그래밍 공부를 시작하였습니다. 
JSP를 이용하여 간단한 게시판을 만들어보는것을 시작으로, 약 2개월간의 독학을 통해 JSP를 이용해 동영상 스트리밍&다운로드 웹사이트를 제작했었습니다. 
수업시간에 이론으로만 배웠던 보안지식들을 프로젝트에 적용시켜봄으로써 더욱 깊이있는 이해를 할 수 있었고, 웹 프로토콜과 JDBC등에 대해서도 깊게 공부할 수 있었던 계기가 되었습니다.  
이후, 웹 프로그래밍에 흥미를 느끼고 SPA기반 웹 페이지를 만들어보고 싶다는 생각에 무작정 JavaScript를 공부하기 시작했습니다. 이후, ES6문법등을 익히고나서 React를 이용해 간단한 웹 게임등을 만들어보았습니다. React에 대한 기초를 학습한 후, Node.js의 기초지식등도 같이 학습하였습니다. 
학습해왔던 React지식과 Node.js의 지식등을 융합해 최종적으로 특정 SNS사이트를 클론 코딩해보고 배포까지 해보는 웹 프로젝트를 완성할 수 있었습니다. React의 프레임워크인 Next를 사용하여 기본 틀을 제작했고, Redux-Saga와 같은 유명 라이브러리등을 이용해 보다 편리하게 개발할 수 있었습니다. 
하지만, 개발과 배포는 완전히 다른 개념의 과정이었고, 오히려 개발을 했던 시간보다 배포를 하기위해 시행착오를 겪었던 시간이 더 오래걸렸을만큼 배포는 쉽지않은 과정이었습니다. 결국 오랜 노력끝에, 시중에 공개되어있는 여러 배포관련 라이브러리등을 이용해 배포까지 끝마칠 수 있었습니다. 
이렇게 JSP와 React를 이용해 웹 사이트를 제작해보고, 나아가 python Flask를 이용해 웹 프로그래밍 기법을 익혀보면서 많은것들을 배우고 학습할 수 있었습니다. 단순히 학교에서 이론으로만 배웠던 개념들이 실제로 어떻게 쓰이는지 알 수 있었고 , 웹 페이지를 제작하는데 필요한 일련의 프로세스등을 명확히 학습할 수 있었습니다. 
또한, 웹 프로그래밍이라고 해서 단순히 웹에만 한정 되어있는것이 아니라, Database, Network, SW보안 등 다방면의 지식을 학습할 수 있어 한층 더 성장할 수 있었던 계기가 되었습니다.
